# Spotiplus
스포티파이 API를 활용한 서비스로 아래와 같은 기능을 제공하고 있습니다.

- 곡 검색
- 추천곡 검색
- 곡 미리듣기
- 가장 많이 들은 곡/아티스트 목록 제작

Demo : https://spotiplus.netlify.app/

## Stack
### Language
* JavaScript

### Front-end
* React.js
* Redux
* Styled-components

### Back-End
* Spotify Open API

### Deploy
* Netlify
